Небольшой веб сервер на 80 порту, написанный для выполнения задания на pwncollege



Принимает запрос от клиента, в зависимости от того какой запрос имеет разные сценарии



Если запрос POST, то в принимаемом запросе считывает путь, после чего пишет в файл по этому пути (или создает, если его нет), ту нагрузку, которую на себе принёс запрос от клиента

Если запрос GET, то в принимаемом запросе считывает путь, после чего из файла по этому пути читает информацию, которую откидывает клиенту
